🎨 Abdulrahman Portfolio

🚀 Abdulrahman Portfolio is a modern personal portfolio website built with React.js and Vite to showcase my work, skills, and projects as a Front-End React Developer.

The website highlights my experience in building responsive, high-performance web applications with smooth animations and a clean user interface.
## 🔥 Features
⚡ Fast performance powered by Vite

🎭 Smooth animations using GSAP

🎥 Interactive animations with Lottie

📩 Contact form integrated with EmailJS

🎨 Responsive design built with TailwindCSS

🏎️ Smooth scrolling experience using Lenis

## 🛠️ Technologies Used

- **Frontend**: React.js, Vite, TailwindCSS
- **Animation**: GSAP, Lenis, Split-Type
- **Utilities**: EmailJS, React Lottie, React Hot Toast
